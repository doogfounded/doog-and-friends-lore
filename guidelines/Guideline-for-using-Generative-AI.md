# Guideline for using Generative AI
## What is allowed

* Using AI for getting ideas and brainstorming
* Training AI on the repo

## What needs appoval

* Using AI for rough drafts of character descriptions, lore, and bios (must be only 10% or under)
* Using AI for automating tasks in the repo

## What is not allowed
* Using AI for completed character descriptions, lore, and bios
